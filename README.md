# SAP RAP Student Data Application

This repository contains the backend core ABAP class code for a student data management application built using the **SAP RESTful Application Programming (RAP)** framework on SAP BTP Cloud.

## 🛠️ Tech Stack Used:
- **Backend Language:** Core ABAP (`zcl_jagrati_hello`)
- **Framework:** SAP RAP (OData V4 - UI)
- **Data Model:** CDS View Entity
